GDPR cookie consent for Koken in Javascript effectively prevents setting cookies by third party scripts like ad or analytics snippets until an explicit consent is given.

GDPR Cookie Consent 1.0.0

GDPR cookie consent is a Koken plugin, that allows you to seamlessly integrade a cookie manager in your portfolio; making your site GDPR Compliant

Latest version

Latest version can be downloaded from the Koken Community Support forum.

  • Asks the visitor to opt-in on cookies that is used
  • Effectively prevents any cookies to be set if user dont accept them
  • Links directly to your privacy Policy/Cookie declaration
  • Fully customazible message
  • Colors can be edited to better integrate with your current color schema
  • Option to ignore specific cookies
  • Option to restrict to EU locations, or disregards geolocation and display for everyone

Installation

  1. Download the plugin
  2. Upload the "GDPR-Cookie-Consent" folder to your plugin dir
    (</storage/plugins/>)
  3. Enable the plugin
    (Koken admin->Settings->Plugins->GDPR Cookie Consent->Enable)
  4. Type in your Clicky site id.
    (Koken admin->Settings->Plugins->GDPR Cookie Consent->Setup)
  5. And your all set to go
